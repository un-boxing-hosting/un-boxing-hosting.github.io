Music-y Bot is a music bot, though it plays music differently from others. This bot is a fork of DrogoBot.

Music-y Bot commands:

=help

This command gives you the exact same thing this is.

=list

This command shows you all of the music on the bot.

=download (youtube.com link) (file name)

This command downloads a YouTube video as an MP3 format with the given name set.

=play (file name)

This command plays any file in the system.

=ping

This command gives you the ping of the bot.
